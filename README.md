# short_links_redirect_get_usernames
from Pinterest short links to getting Pinterest usernames

The Pinterest Username Lookup tool:
Accepts either:
a single Pinterest short link, or
a CSV file containing multiple short links
Automatically follows each short link and pulls out the creator’s Pinterest username from the full profile URL
Generates a ready-to-use query that can be copied directly into Querybook - PII
Allows SPMs to retrieve all related Pinterest user IDs in one go, instead of looking them up one by one
